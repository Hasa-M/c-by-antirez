## Setup Instructions

### Ubuntu
```bash
sudo apt update
sudo apt install build-essential
```

### macOS
```bash
xcode-select --install
```

### Windows
Install Windows Subsystem for Linux (WSL2).  
**Guide:** https://learn.microsoft.com/en-us/windows/wsl/install
